# Digital Warranty & Bill Vault System

A full-stack web application to store, manage, and track product bills and warranties with smart OCR-based expiry detection and reminders.

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS, Shadcn UI, Framer Motion
- **Backend:** Node.js, Express.js, PostgreSQL, Sequelize
- **OCR:** Tesseract.js
- **Auth:** JWT-based authentication

## Structure

- `frontend/` – Next.js app (UI, dashboard, analytics, profile)
- `backend/` – Express.js API (auth, bills, OCR, reminders)
